Model

Line

Tekan tombol "Line", kemudian left click pada canvas untuk menempatkan kordinat pertama line pada canvas, kemudian left click lagi untuk kedua kali untuk menempatkan kordinat kedua line pada canvas

Square

Tekan tombol "Square", kemudian left click pada canvas untuk menempatkan kordinat pertama square pada canvas, kemudian left click lagi untuk kedua kali untuk menggambar square pada canvas

Rectangle

Tekan tombol "Rectangle", kemudian left click pada canvas untuk menempatkan kordinat pertama rectangle pada canvas, kemudian left click lagi untuk kedua kali untuk menggambar rectangle pada canvas

Polygon

Tekan tombol "Polygon", kemudian left click pada canvas untuk menempatkan kordinat pertama rectangle pada canvas, kemudian left click lagi untuk seterusnya untuk memasukkan kordinat sesuai keinginan, kemudian tekan tombol "Enter" untuk menginput kordinat terakhir (kordinat mouse pada saat itu) pada polygon sekaligus menggambarkan polygon pada canvas.

Action

Select Vertice

Tekan tombol "Select Vertice", kemudian pilih vertive yang ingin diubah posisinya dengan left click pada vertice yang bersesuaian, kemudian left click lagi ke kordinat yang diinginkan.

Change Color

Pilih color yang diinginkan pada colr picker, kemudian tekan tombol "Change Color" dan tekan pada area bangun yang ingin diganti warnanya. (untuk mengganti warna line, dapat dengan menekan salah satu verticenya)

File

Menyimpan Gambar

Tekan tombol "Save". Browser akan secara otomatis mendownload file berkstensi .json yang menyimpan model-model yang sedang ada di canvas.

Membuka File

Tekan tombol "Choose file" untuk memilih file yang diinginkan. Kemudian tekan tombol load untuk menampilkan model dari file ke atas canvas.
